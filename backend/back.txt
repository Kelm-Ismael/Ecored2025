GUIA
1. iniciar proyecto desde carpeta backend
    cd backend
    npm init -y
    npm install express dotenv mysql2
    npm install --save-dev nodemon

2. crear .env
    PORT=3000

    DB_HOST=localhost
    DB_USER=tu_usuario
    DB_PASS=tu_contraseña
    DB_NAME=nombre_de_tu_base_de_datos
